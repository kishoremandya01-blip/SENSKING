## HOW TO DEPLOY (Step by Step)

### Step 1 — Upload to GitHub
1. Go to github.com → Sign up free
2. Click "New Repository" → name it "sensking"
3. Click "uploading an existing file"
4. Drag ALL these files (keep folder structure)
5. Click "Commit changes"

### Step 2 — Deploy on Vercel (Free)
1. Go to vercel.com → Sign up with GitHub
2. Click "Add New Project"
3. Select your "sensking" repository
4. Click "Deploy" — done in 60 seconds!
5. You get a free link: sensking.vercel.app

### Step 3 — Connect Your Domain
1. Buy sensking.in from godaddy.com/in (~₹700/year)
2. In Vercel → your project → Settings → Domains
3. Add "sensking.in"
4. Vercel shows you 2 DNS records to copy
5. Go to GoDaddy → DNS Settings → paste those records
6. Wait 10–30 minutes → your site is live at sensking.in!

### Step 4 — Google AdSense (Earn Money)
1. Go to google.com/adsense
2. Sign up with your Google account
3. Add your site URL (sensking.in)
4. They review in 1–2 weeks
5. Once approved, paste their <script> code into index.html before </body>

## ADD NEW DEVICES
Open js/main.js and find the DEVICES object.
Add a new device like this:
```
your_device_key: {name:"Device Name", ram:6, tier:"mid", base:58, screen:6.5},
```
Tier options: "low", "mid", "high"
Base range: 40 (budget) to 75 (flagship)
